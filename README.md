Use it to:
- keep startup tasks in one place
- turn goals into clear next steps
- track sales and marketing work
- review key numbers
- move faster with less manual work

### The file will not open
- right-click the file

### Windows blocks the app
- select **More info**

### The app opens, but nothing loads
- check your internet connection
- close the app
- open it again
- restart your PC if needed

### The text looks too small
- use your browser zoom or Windows display settings
- set display scale to 125% or 150%
